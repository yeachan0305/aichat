## <h2 id="1">1. 개요</h2>

- **대화형 캐릭터 생성** - 사용자가 원하는 봇을 생성 할 수 있습니다.

- **캐릭터와의 대화** - 캐릭터 데이터를 기반으로, AI 언어 모델을 이용해 실시간으로 해당 캐릭터와 대화하는 경험을 제공합니다.

- **커스텀 가능한 대화설정** - 사용자가 원하는 언어 모델이나, 응답 방식 등 세부적인 대화 설정을 제공합니다.

- **정보 공유 게시판** - AI 채팅에 대해 정보를 공유 할 수 있는 게시판 입니다.
